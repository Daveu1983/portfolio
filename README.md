# Personal Portfolio Website

This is a personal portfolio website built with Next.js and Tailwind CSS. It serves as a platform to showcase projects, articles, and other relevant information. It was built using spec-kit

## Technologies Used

*   **Next.js**: React framework for building server-side rendered and static web applications.
*   **Tailwind CSS**: A utility-first CSS framework for rapidly building custom designs.
*   **Spec-Kit**: Used for AI project specification and development. [spec-kit](https://github.com/github/spec-kit)
